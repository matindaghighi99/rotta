ROTTA — STANDALONE HTML WEBSITE

Files:
- index.html: page markup
- styles.css: complete responsive design
- script.js: scroll-controlled video and interactions
- media/: video, poster, and supporting images

To preview locally:
1. Keep all files in the same folder structure.
2. Start any basic local web server in this folder.
3. Open the server address in a modern browser.

For hosting, upload the entire folder contents to the public/root directory.
The scroll film must remain at media/Bag-scrub.mp4 (all-keyframe encode; if the
hero video is ever replaced, re-encode the same way) with its poster at
media/Bag-poster.jpg.
